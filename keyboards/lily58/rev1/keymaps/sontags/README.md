# Sontags Keymap for Lily58

A QWERTY layout with home-row mods for the Lily58 split keyboard.

## Build Commands

```bash
# Compile
qmk compile -kb lily58/rev1 -km sontags

# Or using make
make lily58/rev1:sontags
```

Configure the userspace overlay if not already done:

```bash
qmk config user.overlay_dir="/path/to/qmk_userspace"
```

*Home-row mods (hold for modifier, tap for letter):*

| Key | Tap | Hold |
|-----|-----|------|
| A | a | Shift |
| S | s | Alt |
| D | d | Ctrl |
| F | f | GUI |
| J | j | GUI |
| K | k | Ctrl |
| L | l | Alt |
| ; | ; | Shift |

- `G` and `H`: Tap for letter, hold for Layer 4

**Space key** (layer-tap): Tap for Space, hold for Layer 3 (Mouse). Double-tap and hold for repeating space (via `QUICK_TAP_TERM`).

## Configuration

- **Tapping term**: 200ms
- **Quick tap term**: 200ms (enables double-tap-hold for key repeat)
- **Permissive hold**: Enabled
- **Mouse keys**: Enabled
- **OLED display**: Enabled
- **LTO**: Enabled (smaller firmware size)
- **Converter**: rp2040_ce

### RGB Effects

The Lily58 has RGB underglow with the following effects enabled:
- Breathing
- Rainbow Mood
- Rainbow Swirl
- Snake
- Knight
- Christmas
- Static Gradient
- RGB Test
- Alternating
- Twinkle
